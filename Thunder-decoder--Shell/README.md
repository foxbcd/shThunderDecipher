# thunderdecoder-sh
将下载链接转换为真实链接的工具。

# 支持
- 迅雷链接thunder
- 旋风链接qqdl
- 快车链接flashget
- 飞速链接fs2you
- eDonkey2000Network链接ed2k

# 用法
```
chmod +x thunderdecoder.sh
```
```
./thunderdecoder.sh LINK #转换为真实链接
./thunderdecoder.sh -ac LINK #转换为支持到所有形式到链接
```
